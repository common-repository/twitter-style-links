This plugin allows you to add Twitter style links to comments.

== Description ==

If you write something like "@nick, how are you?" this plugin will search for @nick's url in the comment list of the current post and will convert his/her name to a link.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
